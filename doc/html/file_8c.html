◆ delete_record()

int delete_record ( tipofile_t  tipo)

Elimina il record dal file specificato se esiste.

In base al file specificato dal parametro d'ingresso, visualizza l'interfaccia grafica per inserire i dati dei record e se i dati inseriti sono consistenti elimina dal file giusto il record con l'ID inserito.
Ritorna 0 se la cancellazione e' avvenuta con successo, -1 se i dati inseriti sono inconsistenti o non e' stato possibile trovare il record con l'ID inserito.
Se i file non esistono, questi verranno automaticamente creati.
Inserisce il record di default se necessario nel file dipendenti.txt.

◆ incremental_add_prodotti()

int incremental_add_prodotti ( )

Incrementa o diminuisce il campo quantita' del record con l'ID specificato, se esiste.

Non si puo' decrementare piu' di quello che c'e' in magazzino, se si fa', il record non viene modificato e compare un messaggio di errore.
Se il file prodotti.txt non esiste viene creato automaticamente.
Ritorna 0 se la modifica e' avvenuta correttamente, -1 se si e' tentato di decrementare piu' di quanto consentito, o i dati inseriti sono inconsistenti.

Returns
int

◆ set_novita()

void set_novita ( )

Salvascrive il messaggio inserito al posto di quello presente nel file novita.txt.

Il messaggio deve essere lungo al massimo 400 caratteri (e' escluso il carattere ~).
Il messaggio viene considerato fino al carattere ~ o fino a quando non si e' raggiunto i 400 caratteri.
Il messaggio viene salvascritto solo se si inserisce qualcosa oltre al simbolo ~ .
